Contacts manager for your terminal.

General actions:
  contacts init           Initialize the contacts store
  contacts [ls]           List all contacts (default)
  contacts table          Print contacts info as a table
  contacts help           Show this help

Managing contacts:
  contacts add CONTACT    Add a contact
  contacts rm CONTACT     Delete a contact
  contacts mv OLD NEW     Rename a contact
  contacts edit CONTACT   Edit a contact interactively
  contacts show CONTACT   Edit a contact interactively

Server:
  contacts server         Starts the web server

Git commands:
  contacts git COMMAND    Execute a custom git command
